Data source:
Hjort et al. (2023), SNA datasets from invasive Bombus terrestris on Tasmania, Australia

Processing:
- SNP data converted to PLINK format
- Pairwise genetic distance matrix computed using PLINK (--distance)
- Coordinates converted from UTM to latitude/longitude

Note:
This dataset is a processed derivative of the original publication.
For raw data, see: [https://datadryad.org/dataset/doi:10.5061/dryad.8931zcrxc]
